# 🚀 AutoStream AI Agent

AI-powered conversational agent for AutoStream SaaS built using LangGraph, RAG, and a local LLM (Ollama).

---

## ✨ Features

* Intent detection (greeting, pricing, high-intent)
* RAG-based responses using knowledge base
* Lead capture workflow (name, email, platform)
* Stateful conversation (LangGraph)

---

## ▶️ Run

```bash
pip install -r requirements.txt
ollama pull mistral
python app.py
```
